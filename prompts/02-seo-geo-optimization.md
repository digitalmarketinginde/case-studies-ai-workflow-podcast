# 🚀 Prompt System: SEO + GEO Optimization
## YouTube, Spotify & Apple Podcasts Content Package

> **Goal:** Optimize every episode for human search AND AI-driven discovery (GEO).
> Run these 5 prompts in sequence after receiving the transcript from Gling AI.

---

## INPUT REQUIRED BEFORE STARTING

Prepare these before running any prompt:
- Full episode transcript (exported from Gling AI)
- Episode topic in one sentence
- Any CTA or offer to include (e.g. "Subscribe", "Book a consultation")
- Previous best-performing title (for Prompt 1 reference)

## Prompt 1 — SEO-Optimized YouTube Title

    You are a YouTube SEO strategist specializing in Ukrainian-language marketing content.

    Analyze this podcast transcript: [PASTE TRANSCRIPT]

    Task: Generate an SEO + CTR-optimized YouTube title.

    Rules:
    - Maximum 60 characters (YouTube truncates beyond this in search results)
    - Must include the primary keyword naturally — not forced
    - Must trigger curiosity or emotional response without being clickbait
    - Must be in Ukrainian
    - Must be specific — no vague generic phrases

    Deliver:
    1. RECOMMENDED TITLE — best balance of SEO + CTR
    2. EMOTIONAL VARIANT — triggers strong feeling (fear, curiosity, surprise)
    3. SEO VARIANT — primary keyword in first 3 words
    4. BOLD CLAIM VARIANT — strong opinionated statement

    Character count for each variant must be shown.

    EXAMPLE OF EFFECTIVE FORMAT THAT WORKED:
    "Чому 80% маркетологів витрачають бюджет даремно" — 47 chars
    Result: high CTR because it challenges the viewer's identity directly.

    Analyze what made this work and apply the same logic to the new episode.

    2–3 lines of keyword-rich text in Ukrainian.
    Reads naturally but covers semantic keyword variations.

    Language: Ukrainian throughout
    Total length: 300–500 words

## Prompt 4 — GEO Enhancement Layer

    Review the YouTube description generated in Prompt 2.

    Apply GEO optimization — Generative Engine Optimization.
    Goal: this episode should appear as a cited source when someone asks
    an AI assistant (ChatGPT, Perplexity, Google AI Overview) about [TOPIC].

    Improve the description by:

    1. Adding entity-rich language:
       Mention specific tools, methodologies, frameworks, and named concepts
       that AI systems use to categorize and surface content.

    2. Adding a "What this episode answers" block:
       3–5 bullet points in the format of real user questions.
       Example: "Як зменшити витрати на виробництво контенту без втрати якості?"
       These should be questions your target audience types into AI assistants.

    3. Ensuring quotability:
       Add 1–2 short, specific, standalone insight sentences that an AI
       could extract and quote directly as an answer.
       These should be concrete facts or strong positions — not vague claims.

    4. Removing vague language:
       Replace phrases like "багато переваг", "ефективний підхід",
       "сучасні методи" with specific named techniques and real numbers.

    Output: revised full description with GEO layer applied.
    Language: Ukrainian

## Prompt 5 — Spotify & Apple Podcasts Description

    You are writing a podcast description for audio-first platforms.

    Source: YouTube description from Prompt 2 (already approved)
    Platforms: Spotify + Apple Podcasts (same text used for both via RSS)

    Write a shorter, audio-optimized version following this structure:

    LINE 1 — Hook sentence:
    Must work without visuals — the listener cannot see a thumbnail.
    Must create audio-specific curiosity: "У цьому епізоді ви дізнаєтесь..."
    Maximum 120 characters.

    PARAGRAPH 1 — Core value (60–80 words):
    What specific problem does this episode solve?
    What will the listener be able to DO after listening?
    Include 2–3 keywords for Spotify/Apple search algorithm.
    No timestamps (irrelevant for audio).

    PARAGRAPH 2 — About the podcast (30–40 words):
    One consistent paragraph used across all episodes.
    Describe: podcast name, topic, host expertise, publishing cadence.

    CTA LINE:
    One clear action. Example:
    "Підписуйтесь, щоб не пропустити новий епізод щотижня."

    Total length: 150–250 words
    Language: Ukrainian
    Tone: direct, expert, conversational — no AI-sounding phrases

## Quality Control Checklist

Before publishing, verify all outputs:

- [ ] Title is under 60 characters and tested for CTR
- [ ] First 2 lines of YouTube description compelling without expanding
- [ ] Primary keyword in title + first sentence of description
- [ ] Timestamps cover the full episode structure
- [ ] Tags total under 500 characters
- [ ] GEO layer includes at least 3 specific entities and 2 quotable insights
- [ ] Spotify description works without any visual context
- [ ] No vague AI phrases: "ефективний", "сучасний", "комплексний підхід"
- [ ] Every CTA is specific and actionable
